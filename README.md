# Mali-G52 GPU Clock Fix - Moto G24 🚀

Este módulo Magisk foi desenvolvido para travar o clock da GPU **ARM Mali-G52** no máximo, eliminando quedas de FPS e otimizando o desempenho bruto no **Motorola Moto G24** (Helio G85).

---

## ⚠️ AVISO DE SEGURANÇA (PERFORMANCE EXTREMA)
**O USO DESTE MÓDULO É POR SUA CONTA E RISCO.**
* **Calor:** Este mod força o hardware ao limite máximo de frequência.
* **Cooler Externo:** É **ALTAMENTE RECOMENDADO** o uso de um cooler externo (fan) durante sessões longas de jogo.
* **Responsabilidade:** Não me responsabilizo por degradação de hardware ou bootloops. Use com consciência.

## 🛠️ Instalação e Combo Recomendado

Para que o sistema não reduza o clock por causa do calor (Thermal Throttling), o ideal é usar este módulo junto com um desativador térmico:

1. Instale o módulo **Kreapic-Thermal-Disable** do @mahisataruna.
   - [Baixar Kreapic Module](https://github.com/mahisataruna/Kreapic-Disable-Thermal/tree/v1.0)
2. Instale este módulo via Magisk.
3. Reinicie o dispositivo.
