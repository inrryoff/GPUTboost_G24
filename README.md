# Mali-G52 GPU Clock Fix - Moto G24 🚀

Este módulo Magisk foi desenvolvido para travar o clock da GPU **ARM Mali-G52** no máximo, otimizando o desempenho em jogos e tarefas pesadas no **Motorola Moto G24** (Helio G85).

---

## ⚠️ AVISO DE SEGURANÇA (IMPORTANTE)
**O USO DESTE MÓDULO É POR SUA CONTA E RISCO.**
* **Superaquecimento:** Este mod força o hardware ao limite.
* **Hardware:** O uso prolongado sem resfriamento pode causar danos permanentes.
* **Bootloop:** Existe o risco do sistema não iniciar em alguns dispositivos.
* **Cooler Externo:** É **OBRIGATÓRIO** o uso de um cooler externo durante o uso intenso.

---

## 📱 Compatibilidade
* **Testado em:** Moto G24.
* **Chipset:** MediaTek Helio G85.
* *Se testar em outro aparelho com Helio G85, por favor, reporte o resultado!*

---

## 🛠️ Instalação e Recomendações

Para evitar que o sistema derrube o desempenho (Thermal Throttling), é altamente recomendável utilizar este módulo em conjunto com o desativador térmico abaixo:

1. Instale o módulo **Kreapic-Thermal-Disable** do @mahisataruna.
   - [Baixar Kreapic Module](https://github.com/mahisataruna/Kreapic-Disable-Thermal/tree/v1.0?tab=readme-ov-file)
2. Instale este módulo via Magisk.
3. Reinicie o dispositivo.
